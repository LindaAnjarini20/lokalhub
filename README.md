## Getting Started
To get started with LokalHub, you'll need to:  
1. Clone the repository:  
   ```bash
   git clone https://github.com/LindaAnjarini20/lokalhub.git
   ```  
2. Install dependencies:  
   ```bash
   cd lokalhub
   npm install
   ```  
3. Run the project:  
   ```bash
   npm start
   ```
